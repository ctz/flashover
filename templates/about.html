Technology

Flashover is built with:

  • jQuery, jQuery File Upload and Twitter Bootstrap take care of the front-end.
  • web.py glues everything together.
  • MySQL provides metadata storage.
  • pyswf does the dirty work (with lots of extensions).
  • PIL does image processing.

Similar services

  • Google Swiffy performs conversion of SWF movies to HTML5 by converting the SWF to a JSON representation which is interpreted and executed by a javascript library.
  • swftools contains a number of tools which do equivalent things to flashover.
There are many more!
